# Release 1.0.0 - GenDev 1.0 Production Release

Status: Production candidate; publication pending required gates
Date: 2026-07-11
Supersedes: 0.5.0-operational-coherence
Publication tag: planned `v1.0.0`
GitHub Release title: `GenDev 1.0 Production Release`
Execution log: `docs/resources/evolution/1.0.0-production-release-execution-log.md`
Adoption guide: `docs/resources/releases/1.0.0-adoption.md`

## Summary

GenDev 1.0 promotes the operationally coherent 0.5 methodology into a production
release for AI-assisted engineering practitioners. The release focuses on
truthful release identity, agent-operable workflows, coherent project
scaffolding, full lifecycle validation, production adoption guidance, and a
publication process gated by local validation, remote CI, branch protection,
independent review, tag creation, and GitHub Release publication.

## Publication Gates

1. Local validation passes.
2. Remote CI is configured and observed green.
3. Branch protection is configured before publication.
4. Independent 1.0 coherence review records a non-pending result.
5. Release notes, release index, lifecycle registry, generated contract, README,
   adoption guide, and practitioner guide agree on release identity.
6. Annotated tag `v1.0.0` is created at the reviewed release commit.
7. GitHub Release `GenDev 1.0 Production Release` is published.

## Known Residuals

Residuals are not accepted for publication unless recorded in the execution log
with an explicit human disposition.
