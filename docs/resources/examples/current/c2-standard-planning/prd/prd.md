# PRD

Status: Accepted
project: standard-planning-example

REQ-C2-001: Contract inventory list.

EARS acceptance criterion: When an authorized operator opens the contract inventory, the system
shall show vendor name, owner, status, renewal date, and notice deadline for each stored contract.

REQ-C2-002: Notice-window filter.

EARS acceptance criterion: When an authorized operator applies the 60-day notice filter, the system
shall show only contracts whose notice deadline is within the next 60 days.

Unwanted behavior: If an unauthorized user requests contract inventory data, the system shall deny
access and shall not disclose contract records.
