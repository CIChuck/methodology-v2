# Minimal SaaS Product Example

> **Phase-loop exemption.** This example predates the G5 phase loop (the
> `G5.x` checkpoints and the six-artifact phase set). Its artifacts illustrate
> earlier single-phase structure and do not demonstrate the phase plan,
> phase-loop checkpoints, or phase exit tests. A worked phase-loop example is
> deferred to a later release. Examples are non-authoritative.


Status: Example Only

This directory is a small worked example of the methodology applied to a simple SaaS product:
a team task tracker with project boards, tasks, comments, and audit-friendly status changes.

It is not active project authority. Do not use these files as implementation evidence for a cloned
project unless they are copied into `docs/project/` and reconciled with that project's actual code,
tests, and decisions.

team coordination requires more detail.
